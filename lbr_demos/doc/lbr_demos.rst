lbr_demos
=========

.. note::
    Some demos require a real robot. Make sure you followed :doc:`Hardware Setup <../../lbr_fri_ros2_stack/doc/hardware_setup>` first.

.. warning::
    On the real robot, do always execute in ``T1`` mode first.
